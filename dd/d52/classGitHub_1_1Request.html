Detailed Description

GitHub api actions.

Class contains a convenient set of actions which can be executed on GitHub's api.

Before one can use it a connection with github needs to be established. Use GitHub::QtBackend::Api or GitHub::CurlBackend::Api to construct an GitHub::IConnection object.

All methods return a response in json format.
